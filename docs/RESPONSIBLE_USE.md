# Responsible Use

MIRROR is intended for authorized security evaluation of agentic RAG systems.

Use this code only when you have permission to test the target system. Do not use it to attack third-party systems, evade monitoring, or deploy harmful content.

This public package avoids shipping private payload records and secrets. The reproducibility scripts are designed to work with payload-free run records and approved public artifacts.

If you extend this package, hold the same bar: no credentials, no private target data, and clear documentation for every included artifact.
